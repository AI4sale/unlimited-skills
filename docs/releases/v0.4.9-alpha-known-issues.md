# v0.4.9-alpha Known Issues

`v0.4.9-alpha` is an alpha release for the MCP profile rollout simulator and
policy doctor. The known boundaries are intentional:

- No E17 audit replay.
- No E12B warm-cache implementation.
- No profile activation.
- No trust-store mutation.
- No hosted trust fetch.
- No registry sync.
- No production signing keys.
- No private key storage.
- No OAuth.
- No MCP resources.
- No MCP prompts.
- No production rollout.
- No PyPI publication.
- No live billing.
- Rollout plans are preview evidence only; they do not start the gateway,
  spawn upstreams, write audit rows, or apply policy.
