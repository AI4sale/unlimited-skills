# v0.3.4-alpha Upgrade Notes

Upgrade from a GitHub clone:

```bash
git fetch --tags origin
git checkout <v0.3.4-alpha-tag-or-release-branch>
python -m pip install -e ".[all]"
unlimited-skills --version
unlimited-skills reindex
unlimited-skills doctor --json
unlimited-skills plan status --json
unlimited-skills billing status --json
```

PyPI is not the supported `v0.3.4-alpha` distribution path.
GitHub clone remains the official alpha distribution path.

## From v0.3.3-alpha

- Existing local libraries should remain in place.
- Existing private team packs remain under `registry/private/<pack_id>`.
- `plan status`, `plan explain`, and `plan doctor` remain local/cache-only.
- `billing status` and `billing doctor` are new local/cache-only diagnostics.
- `billing refresh` requires registration and hosted service support for `/v1/hub/billing-status`.
- `past_due`, `suspended`, and `expired` are now first-class lifecycle denial reasons across plan, billing, and private-pack diagnostics.
- Support bundle now includes a redacted billing lifecycle summary.
