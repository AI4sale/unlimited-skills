# v0.6.3-alpha Upgrade Notes

Status: release execution package.

After upgrading, existing local search, feedback, and learning-summary commands
remain available. v0.6.3 adds runnable local Learning Loop tier commands.

Install after the release is published:

```powershell
python -m pip install --upgrade "unlimited-skills==0.6.3"
unlimited-skills --version
```

Expected version output after publish:

```text
unlimited-skills 0.6.3
```

## Free/Base

```powershell
unlimited-skills learning doctor
unlimited-skills improvement-candidates
unlimited-skills apply-candidate --dry-run <candidate-id>
```

To create candidate input, record explicit local feedback:

```powershell
unlimited-skills feedback record <skill-name> --verdict wrong
unlimited-skills feedback record <skill-name> --verdict missed
unlimited-skills feedback record <skill-name> --verdict rejected
```

## Above-Free Local Artifacts

```powershell
unlimited-skills learning export --out registered-learning-export.json --json-status
unlimited-skills learning team-rollup --input registered-learning-export.json --out team-learning-rollup.json --json-status
unlimited-skills learning admin-export --input team-learning-rollup.json --json business-learning-admin.json --csv business-learning-admin.csv
unlimited-skills learning evidence-pack --input business-learning-admin.json --out enterprise-learning-evidence-pack
unlimited-skills learning verify-evidence-pack --input enterprise-learning-evidence-pack --json
```

These are local file commands. They do not upload, sync, create hosted
dashboards, enable telemetry, mutate skills, or apply candidates automatically.

No hosted dashboard, live sync, SSO/SCIM, hosted governance, billing, telemetry,
signature-enforced claim, docs-only tier claim, automatic skill improvement
claim, v0.6.4 release claim, marketplace submission, or #119/E19 inclusion is
part of v0.6.3-alpha.

## Compatibility

v0.6.2 router-health tier-debt commands remain available and are checked by the
v0.6.3 smoke verifier as compatibility evidence. They are not the v0.6.3 Learning
Loop value frame.
