# v0.3.0-alpha

Status: release candidate publication gate.

`v0.3.0-alpha` is the productionization alpha for Unlimited Skills. It keeps the MIT local core usable without registration while adding the in-review registered stack for production-shaped registry diagnostics, managed Enterprise Skill Lock sync, packaging hardening, and release evidence.

## Highlights

- Package version is `0.3.0`.
- GitHub clone remains the supported alpha distribution path.
- PyPI is explicitly deferred for this alpha because repo assets are still required.
- Managed Enterprise Skill Lock policy sync is implemented on the public client and covered by fixture E2E.
- Managed policy removal now refuses to delete unmanaged local policies.
- v0.3 release smoke includes managed policy sync E2E.
- Packaging smoke verifies required router skills, installers, schemas, docs, and CLI version.

## Security Boundary

- MIT local commands remain unregistered.
- `unlimited-skills serve` remains unregistered.
- Hosted catalog, updates, team sync, registered enhancement, Local Skill Hub registry sync, and managed policy sync remain registration-gated.
- Full catalog distribution remains disabled.
- Local Skill Hub remains allowlist-only and does not execute skills.
- Hosted remote manifests must include valid signed manifest envelopes.
- Hosted collection archives and enhancement scripts are SHA256-verified before use.
- Archive bytes are not independently cryptographically signed in this alpha.
- Private registry skill bodies, signing private keys, hosted tokens, and device private keys are not committed to the public repo.
