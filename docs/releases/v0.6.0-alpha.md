# v0.6.1-alpha

Status: contract-freeze and local ROI receipt packaging gate. PyPI publication
is blocked until Trusted Publishing and the post-publish verifier pass.

`v0.6.1-alpha` freezes the public-alpha CLI/stdout/privacy contracts documented
in [v0.6 contract freeze](v0.6-contract-freeze-spec.md) and ships the
privacy-safe local ROI receipt runtime defined in
[v0.6 local ROI receipt spec](v0.6-local-roi-receipt-spec.md).

## What Changed

- Package, Claude plugin, and marketplace metadata versions are `0.6.1`.
- Hotfix over the untagged `0.6.0` PyPI upload: `learning-summary --events
  --json` is now accepted explicitly and preserves aggregate-only JSON output.
- The public-alpha contract freeze documents stable behavior for install,
  quickstart, suggest/search/view/use, MCP savings/install dry-runs, feedback
  prepare, learning-summary event aggregation, local event privacy, and PyPI
  Trusted Publishing.
- Adds privacy-safe local ROI receipt:
  - `unlimited-skills roi receipt`;
  - `unlimited-skills roi receipt --format json`;
  - `unlimited-skills roi receipt --since 7d`;
  - `unlimited-skills roi receipt --out roi-receipt.md`.
- Receipt output is local-only and aggregate-only.
- Receipt output is not telemetry, not a benchmark guarantee, and not a paid ROI
  promise.
- APIs may still change before 1.0 only where documented.
- `scripts/run-v060-alpha-package-smoke.py` proves clean-install behavior for
  quickstart, suggest, MCP savings, MCP install dry-run, feedback prepare,
  learning-summary, and ROI receipt Markdown/JSON.
- `scripts/verify-v060-alpha-publication.py` gates metadata, release docs,
  manifest requirements, clean-install smoke, ROI receipt boundaries,
  prepublish state, and post-publish PyPI smoke before tag guidance is emitted.

## Boundaries

- #119 / E19 is not included in this release unless explicitly approved later.
- No marketplace submission is performed by this release gate.
- No hosted readiness, no team readiness, no enterprise readiness, paid CTA, payment
  link, support/SLA promise, delivery promise, ROI guarantee, or universal
  savings promise is added.
- No telemetry, analytics SDK, tracking pixel, automatic upload, hosted query
  forwarding, private social scraping, private user data collection, prompt
  collection, tool input collection, or tool output collection is added.
- The tag `v0.6.1-alpha` remains blocked until `unlimited-skills==0.6.1` is
  uploaded to PyPI and `verify-v060-alpha-publication.py
  --package-availability published` passes on the selected release commit.
