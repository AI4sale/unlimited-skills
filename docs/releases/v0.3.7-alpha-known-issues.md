# v0.3.7-alpha Known Issues

`v0.3.7-alpha` is a developer preview.

- Hosted catalog feedback requires registration and the private registry feedback API deployment.
- Release tests use fixture mode and do not call production hosted services.
- Feedback signals are quality inputs for maintainers; they are not automatic telemetry and are not used for automatic marketplace ranking.
- Full catalog distribution remains disabled. Registered catalog delivery remains governed by signed metadata, allowlists, and review status.
- PyPI remains deferred for this alpha. GitHub clone is the documented distribution path.
- Support bundles intentionally summarize feedback status instead of including raw feedback payloads.
- Catalog feedback must not be used to send prompts, task text, skill bodies, customer data, local paths, repo paths, secrets, proofs, or private keys.
