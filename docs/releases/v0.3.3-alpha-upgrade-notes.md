# v0.3.3-alpha Upgrade Notes

Upgrade from a GitHub clone:

```bash
git fetch --tags origin
git checkout <v0.3.3-alpha-tag-or-release-branch>
python -m pip install -e ".[all]"
unlimited-skills --version
unlimited-skills reindex
unlimited-skills doctor --json
unlimited-skills org status --json
unlimited-skills private-packs doctor --json
```

PyPI is not the supported `v0.3.3-alpha` distribution path.

## From v0.3.2-alpha

- Existing local libraries should remain in place.
- Existing private team packs remain under `registry/private/<pack_id>`.
- `private-packs access-check <pack_id>` can now explain hosted entitlement, team, agent, channel, revocation, policy, or service availability failures without downloading the pack.
- `org status` can read the local governance cache without network access.
- `org status --refresh` requires a registered installation and hosted service support.
- Support bundle now includes redacted org/team/private-pack entitlement summaries.

## Verify After Upgrade

```bash
python scripts/run-private-team-pack-cross-repo-e2e.py --fixture-mode --temp-home --json
python scripts/run-v0.3.3-alpha-org-team-governance-smoke.py
python scripts/verify-v0.3.3-alpha-org-team-governance.py
```

Registered hosted flows still require a registered installation, hosted token, signed device proof, trusted signed manifests, and registry-side authorization.
