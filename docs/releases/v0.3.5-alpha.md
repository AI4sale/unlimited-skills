# v0.3.5-alpha Community Catalog Integration Gate

Status: in review.

This release gate integrates the private registry community submission review workflow with the public community catalog client v2.

## Scope

- Registry submission states: draft, uploaded, pending_review, changes_requested, approved, published, rejected, withdrawn, deprecated, retired.
- Registry validation for `SKILL.md`, metadata, license/source, secrets, private paths, binaries, archives, executable scripts, policy bypass instructions, and private customer data.
- Signed canary publication for approved community submissions.
- Public client channel-filtered community list.
- Public client submission status, withdraw, and review-notes commands.
- Public client preview/install restriction to signed approved or published hosted community items.

## Out Of Scope

- No skill execution.
- No billing, revenue share, storefront, checkout, payment links, invoices, refunds, cards, or bank data.
- No full catalog distribution.
- No stable promotion by default.
- No production hosted calls in release tests.
