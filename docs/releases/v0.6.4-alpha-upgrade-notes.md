# v0.6.4-alpha Upgrade Notes

Status: release execution package.

After publication, install or upgrade with:

```powershell
python -m pip install --upgrade "unlimited-skills==0.6.4"
unlimited-skills --version
```

Expected version output:

```text
unlimited-skills 0.6.4
```

New local Money Saved Meter surfaces:

```powershell
unlimited-skills money-saved meter --json
unlimited-skills money-saved registered-export --help
unlimited-skills money-saved team-rollup --help
unlimited-skills money-saved admin-export --help
unlimited-skills money-saved evidence-pack --help
unlimited-skills money-saved verify-evidence-pack --help
```

The release remains local-first. It does not enable hosted telemetry, automatic
upload, live sync, hosted dashboards, billing reconciliation, marketplace
submission, hosted/team/business/enterprise rollout, or #119/E19 inclusion.
