# v0.6.1-alpha

Status: valid v0.6 public alpha.

`v0.6.1-alpha` is the accepted v0.6 alpha release. It replaces the uploaded but
untagged `0.6.0` PyPI artifact after the published verifier caught that
`learning-summary --events --json` was rejected by the installed `0.6.0` CLI.
v0.6.1-alpha is the accepted v0.6 alpha release.

## Install

```bash
python -m pip install --upgrade "unlimited-skills>=0.6.1"
unlimited-skills quickstart
```

For exact reproduction of this release:

```bash
python -m pip install --upgrade "unlimited-skills==0.6.1"
unlimited-skills --version
unlimited-skills learning-summary --events --json
```
