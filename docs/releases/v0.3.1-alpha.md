# v0.3.1-alpha

Status: final publication gate in review.

`v0.3.1-alpha` stabilizes the published `v0.3.0-alpha` baseline. It does not enable full catalog distribution, publish to PyPI, or gate the MIT local core. The release focuses on release health, install and upgrade evidence, first-run setup, redacted support diagnostics, diagnostics v2, and support readiness.

Published baseline: https://github.com/AI4sale/unlimited-skills/releases/tag/v0.3.0-alpha

Publication manifest: `docs/releases/v0.3.1-alpha.release-manifest.json`

## Scope

- Package version is `0.3.1`.
- Published baseline is `v0.3.0-alpha`.
- GitHub clone remains the supported alpha distribution path.
- PyPI is explicitly deferred for this alpha because repo assets are still required.
- MIT local commands remain unregistered.
- Hosted registry, update, team, Local Skill Hub registry sync, and managed Enterprise Skill Lock sync remain registration-gated.
- Full catalog distribution remains disabled.
- Local Skill Hub remains allowlist-only.
- First-run setup wizard and support diagnostic bundle are included in the public release stack.
- Diagnostics v2 is connected to setup and support workflows.
- Private registry reconciliation remains in review: PR #10 records the canonical 315-skill audit with 105 allowlisted skills, 165 local-install-plan skills, and 45 excluded skills.
- Production-signed registry artifacts remain blocked until a registry operator completes the protected signing ceremony. The default final publication verifier fails in this state unless the release owner explicitly accepts blocked registry signing as a known issue.

Keep this release task in review until the release owner verifies the install, upgrade, docs, known-issues evidence, and production registry signing state.
