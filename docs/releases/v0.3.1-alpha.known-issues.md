# v0.3.1-alpha Known Issues

- PyPI is not the supported distribution path for this alpha. Use a GitHub clone because router skills, installers, schemas, docs, and migration scripts are repo assets.
- Full catalog distribution remains disabled. Local Skill Hub distribution remains allowlist-only.
- Hosted registry features require registration and signed hosted manifests.
- Archive bytes are SHA256-verified and safely extracted, but archive-byte cryptographic signatures are not implemented in this alpha.
- Enterprise Skill Lock is an opt-in policy layer unless a managed enterprise policy is installed.
- Dashboard, billing, SSO/SCIM, and broad organization controls are not part of v0.3.1-alpha.
- Support bundle v1 is metadata-only. It excludes skill bodies, prompts, search queries, env values, tokens, private keys, private local skill names, and local paths by default.
