# v0.4.9-alpha Upgrade Notes

`v0.4.9-alpha` is a public developer-preview release for the MCP profile
rollout simulator and policy doctor. It keeps the distribution path as GitHub
clone/install scripts and does not require hosted services.

## What Changes

- Package and Claude plugin metadata move to `0.4.9`.
- `unlimited-skills mcp profiles rollout-plan|doctor` is documented as the
  release milestone.
- Release smoke now proves the E16 rollout simulator together with the prior
  v0.4.8 managed trust-store gate.

## What Does Not Change

- No profile activation.
- No trust-store mutation.
- No upstream spawn.
- No network access.
- No hosted trust fetch.
- No registry sync.
- No production signing keys.
- No OAuth, MCP resources, or MCP prompts.
- No live billing, PyPI publication, auto-publish, or full catalog
  distribution.

## Verification Before Publishing

```powershell
$sha = git rev-parse HEAD
.\.venv\Scripts\python.exe scripts\run-v049-alpha-release-smoke.py
.\.venv\Scripts\python.exe scripts\verify-v049-alpha-publication.py --expected-sha $sha
```

Codex may create and push `v0.4.9-alpha` only after the final publication
verifier passes on the selected `main` SHA.
