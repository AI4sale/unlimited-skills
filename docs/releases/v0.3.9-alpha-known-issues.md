# v0.3.9-alpha Known Issues

`v0.3.9-alpha` is a developer preview.

- Hosted improvement status requires registration and a compatible private registry deployment.
- Release tests use fixture mode and do not call production hosted services.
- External local-registry E2E requires a local `AI4sale/unlimited-skills-registry` checkout with the v0.3.9 private registry changes.
- Improvement candidates are maintainer workflow metadata. They are not automatic telemetry and they are not used for automatic marketplace ranking.
- Update recommendations are preview-only and do not apply changes.
- Full catalog distribution remains disabled. Registered catalog delivery remains governed by signed metadata, allowlists, review status, and capability checks.
- PyPI remains deferred for this alpha. GitHub clone is the documented distribution path.
- Support bundles intentionally summarize improvement status instead of including raw issue payloads or item ids.
- Skill improvement feedback/status must not be used to send prompts, task text, skill bodies, customer data, local paths, repo paths, secrets, proofs, or private keys.
- v0.4 remains NO-GO until the P0 blockers in `docs/releases/v0.4-readiness-audit.md` are closed.
