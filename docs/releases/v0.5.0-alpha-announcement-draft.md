## Long form (GitHub release notes body)

**unlimited-skills v0.5.0-alpha — first public alpha**

Coding agents waste most of their context window before the first prompt:
every connected MCP server dumps its full tool schemas into every session,
and skill libraries either flood the context or never get used at all.

unlimited-skills is a local-first fix. It keeps a searchable library of
agent skills (267 bundled: ECC + Superpowers packs) and an MCP gateway that
loads one tool's schema only when it is needed.

What this alpha actually does, measured on this repo's frozen eval set:

- A deterministic sub-second `suggest` probe instead of an LLM guessing
  which skill might exist — top-3 hit rate 0.967, zero false positives on
  negative scenarios.
- Tiered ambient suggestions in Claude Code: silence below confidence
  floor, a one-line hint, or a single skill card at high confidence — with
  a kill switch (`UNLIMITED_SKILLS_NO_INJECT=1`).
- MCP gateway standing cost: 90,420 → 1,268 bytes of tool schemas in our
  lab benchmark (98.6% less). Run `unlimited-skills mcp savings` to see the
  numbers from YOUR config — measured locally, uploaded nowhere.
- A 2-minute golden path: `unlimited-skills quickstart` imports the bundled
  packs, runs your first search, and shows your personal savings report.

What this alpha is NOT:

- Not a paid product. There are no paid plans and nothing for sale.
- Not a telemetry funnel. The tool sends nothing anywhere; the learning
  log stays on your disk.
- Not stable. APIs may break between alphas until 0.6; known limitations
  are listed in [docs/releases/v0.5.0-alpha-known-issues.md](v0.5.0-alpha-known-issues.md).

Found friction in your first five minutes? That is exactly the feedback we
want: use the issue templates (first-value feedback, install friction,
skill not invoked, MCP savings report) — see
[docs/feedback.md](../feedback.md).

## Short form (one paragraph, discussions/social)

unlimited-skills v0.5.0-alpha is out — a local-first skill library and MCP
gateway for coding agents. Instead of flooding the context with every tool
schema and skill body, the agent searches first and loads one thing when
needed: sub-second skill lookup (0.967 top-3 on our frozen evals), an MCP
gateway that cut schema overhead 98.6% in our lab benchmark, and a
`quickstart` that shows the savings measured on YOUR machine. Free, open
source, no telemetry, nothing for sale. Feedback via GitHub issues —
especially about your first five minutes.
