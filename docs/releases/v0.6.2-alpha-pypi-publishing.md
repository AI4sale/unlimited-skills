# v0.6.2-alpha PyPI Trusted Publishing

Status: required publish gate for `v0.6.2-alpha`. The release tag stays blocked
until Trusted Publishing uploads `unlimited-skills==0.6.2` and the post-publish
verifier passes.

## PyPI Trusted Publisher Values

| Field | Value |
| --- | --- |
| Project/package name | `unlimited-skills` |
| Owner | `AI4sale` |
| Repository | `unlimited-skills` |
| Workflow filename | `publish-pypi.yml` |
| Environment | `pypi` |
| Package version | `0.6.2` |

No `PYPI_TOKEN`, `TWINE_PASSWORD`, `TWINE_USERNAME`, or local `.pypirc` is used
by the release workflow. Publishing uses GitHub OIDC through PyPI Trusted
Publishing.

## Manual Workflow Inputs

```text
version: 0.6.2
expected_sha: <selected main commit>
confirm_pypi_publish: publish unlimited-skills 0.6.2 to PyPI
```

## Pre-Publish Commands

```bash
python scripts/verify-v06-frozen-contracts.py --json
python scripts/run-v062-alpha-package-smoke.py --json
python scripts/verify-v062-alpha-publication.py \
  --expected-sha "$(git rev-parse HEAD)" \
  --package-availability prepublish \
  --json
```

## Post-Publish Commands

Run only after the PyPI workflow completes:

```bash
python -m pip install --no-cache-dir unlimited-skills==0.6.2
unlimited-skills --version
unlimited-skills quickstart --json
unlimited-skills suggest "Design a REST API for a service" --json
unlimited-skills mcp install --claude-code --dry-run
unlimited-skills mcp savings --json
unlimited-skills feedback prepare --json
unlimited-skills learning-summary --events --json
unlimited-skills roi receipt
unlimited-skills roi receipt --format json
unlimited-skills roi receipt --since 7d
python scripts/verify-v062-alpha-publication.py \
  --expected-sha "$(git rev-parse HEAD)" \
  --package-availability published \
  --json
```

Only the published verifier may unlock the `v0.6.2-alpha` tag. The verifier
invokes `scripts/verify-v06-frozen-contracts.py --json` in both prepublish and
published modes; any frozen-contract drift keeps publish/tag guidance blocked
with owner/action/fallback details.
