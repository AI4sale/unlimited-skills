# v0.6.5 zero-candidate gate audit

Status: C065-GATE-AUDIT-01 audit evidence only. This PR does not implement
O065-RETRIEVAL-FIX-01/02.

## Executive finding

The current delivery path can still give the model zero or too few skill
candidates even when retrieval has candidates in the same non-empty library.

The strongest current failures are:

- non-English prompts hit the lexical-first `suggest` path, miss the floor, and
  deliver no candidate names to `UserPromptSubmit`;
- English prompts with available retrieval candidates can still be suppressed by
  the calibrated floor;
- `UserPromptSubmit` hardcodes `--limit 1`, so it cannot deliver the required
  3-5 candidate family even when direct retrieval has it;
- `suggest` and `search --mode hybrid` do not share one candidate generator.

## Gate matrix

The verifier records these fields for every query:

- `library_skill_count`
- `lexical_search_top_10`
- `vector_search_top_10`
- `hybrid_search_top_10`
- `english_hybrid_search_top_10`
- `suggest_json`
- `suggest_card_json`
- `user_prompt_submit`
- `drop_reasons`

The observed gates are:

- `floor`: `suggest` suppresses below-`DEFAULT_FLOOR=12.0` candidates and reports `below_floor`.
- `lexical-only path`: the hook uses `suggest`, not `search --mode hybrid`, and `suggest` has no `--mode`.
- `limit=1`: the hook hardcodes `--limit 1`, so it cannot deliver a 3-5 candidate family.
- `card threshold`: medium candidates become hints, not cards.
- `high-margin gate`: contested top candidates can downgrade to a hint.
- `missing sidecar`: vector path fails when no local vector sidecar exists.
- `non-English branch`: native-language prompts become `needs_english_query` instead of candidate delivery.
- `timeout`: not observed in this run, but the verifier exposes it when present.

## New invariant

In a non-empty skill library, `UserPromptSubmit` must not inject zero candidates
when any supported retrieval path returns candidates.

The system must prefer noisy top-k candidates over silent failure. The model can
ignore weak candidates; it cannot use candidates it never sees.
