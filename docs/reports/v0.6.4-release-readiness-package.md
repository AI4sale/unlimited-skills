## Decision Summary

Unlimited Skills v0.6.4-alpha is ready to present the local Money Saved Meter as
a reproducible, privacy-safe alpha surface with explicit limits. The release can
claim local counts, measured bytes when source artifacts expose bytes, and
estimate-only token/dollar framing. Release execution is not authorized by this
package alone.

Allowed public claim:

```text
Unlimited Skills v0.6.4-alpha adds a local Money Saved Meter that helps users
measure before/after context savings, inspect a deterministic 100-call fixture,
and reproduce the report on their own machine without uploading private data.
```

Release recommendation: `GO_WITH_LIMITS`.

## Release Decision

`GO_WITH_LIMITS`.

The local Money Saved Meter is honest enough for an alpha release if release
notes and README copy stay inside the allowed claim boundary. It is not a
general ROI guarantee, not a hosted analytics feature, and not a paid-tier
rollout.

Release execution remains blocked until the release operator runs the selected
release flow from refreshed `main`, updates version/tag/package artifacts as a
separate approved step, uses PyPI Trusted Publishing if publishing, verifies a
clean install, and records post-publish evidence.
