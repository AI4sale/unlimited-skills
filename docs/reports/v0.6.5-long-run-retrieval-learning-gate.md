# v0.6.5 Long-Run Retrieval Learning Gate

## Purpose

`C065-LONGRUN-01` adds a release-grade regression gate for the O065 retrieval and learning repairs. The gate simulates a 100-step, 10-phase agent run where the agent asks the router for skills, receives shared candidate-family results, records accepted and wrong outcomes, and later proves learning changes ranking without leaking raw user text.

## Gate

Run:

```bash
python scripts/verify-v065-long-run-retrieval-learning.py --json
```

The verifier uses `tests/fixtures/retrieval/long_run_phases.v1.json` and executes the phase steps through CLI/helper surfaces instead of static JSON validation. It verifies:

- search, suggest, and UserPromptSubmit do not return zero candidates when retrieval has candidates;
- UserPromptSubmit delivers candidate names/hints for relevant queries;
- shared candidate families remain consistent across retrieval surfaces;
- accepted feedback boosts exact and similar later queries;
- accepted feedback boosts a later mixed/Russian-language query through privacy-safe expanded token fingerprints;
- wrong feedback demotes later candidates and exposes `learning_demotion`;
- a manual `search -> view -> use` path learns even when `use` has no explicit `--query`;
- phase boundaries trigger fresh re-querying instead of stale candidate reuse;
- learning logs contain query hashes and token hashes only, with no raw prompt/query, notes text, absolute local path, or skill body.

## Expected Shape

The report must return:

```json
{
  "ok": true,
  "schema_version": "v065-long-run-retrieval-learning-v1",
  "step_count": 100,
  "phase_count": 10,
  "zero_candidate_losses": 0,
  "phase_boundary_requeries": 9,
  "accepted_rank_lift_cases": 3,
  "similar_query_lift_cases": 3,
  "cross_language_lift_cases": 1,
  "wrong_skill_demotion_cases": 2,
  "manual_filesystem_skill_walk_detected": false,
  "privacy_ok": true,
  "failures": []
}
```

Counts may exceed the minimums, but `ok` must be true and `failures` must be empty.

## Non-Goals

This gate does not publish, tag, package, or start v0.6.5 release work. It is a regression verifier for the already merged O065 retrieval and learning repairs.
