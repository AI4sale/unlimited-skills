# v0.6.4-alpha Release Decision Package

Recommendation: `GO_WITH_LIMITS` for release execution package review.

Roadmap ref:
`docs/roadmaps/unlimited-skills/UNLIMITED-SKILLS-PRIVATE-ROADMAP-v0.6.3-to-v0.6.13.md#US-064-005`

The v0.6.4 Money Saved Meter may be released as an alpha for local,
command-based measurement and tier artifacts if all C064-REL-01 gates pass and
Opus returns O064-REL-FINAL-R1 PASS.

Allowed claims:

- local Money Saved Meter;
- local before/after measurement;
- deterministic 100-call fixture;
- exact counts remain counts;
- measured bytes remain measured bytes;
- tokens are estimates;
- dollars are disabled by default;
- local-only privacy posture;
- Free, Registered, Team, Business, and Enterprise tier artifacts are runnable
  local command surfaces.

Blocked claims:

- exact token savings;
- exact money savings;
- guaranteed bill reduction;
- provider billing reconciliation;
- hosted telemetry-backed savings;
- live hosted dashboard;
- automatic upload or sync;
- persistent hosted meter state;
- marketplace submission;
- hosted/team/business/enterprise rollout;
- #119/E19 inclusion.

Required gates:

- `python scripts/verify-v064-money-saved-tier-smoke.py --json`;
- `python scripts/verify-v064-alpha-release-execution.py --json`;
- `python scripts/run-v064-alpha-package-smoke.py --json`;
- `python scripts/verify-v06-frozen-contracts.py --json`;
- `python scripts/verify-feedback-report-boundaries.py`;
- focused and full pytest suites;
- `git diff --check`.

Release execution is deferred to the release operator. This package does not
publish, tag, yank, create a GitHub release, submit marketplace listings, or
start hosted rollout.

Short boundary wording for release review: no exact money, no exact token, and
no guaranteed bill reduction.
