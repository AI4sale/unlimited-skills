# Product Editions

Unlimited Skills has a local MIT core and registration-gated hosted services. Registration must not be required for local Community Core commands.

| Edition | Status | Registration | Included |
| --- | --- | --- | --- |
| Community Core | MIT public repo | No | Local `search`, `list`, `view`, `where`, `use`, `feedback`, `reindex`, `vector-reindex`, `adapt`, `adapt-one`, `adapt-next`, `apply-adaptation`, `serve`, installers, migration scripts, native sync, local learning logs, bundled repo assets, and public self-update. |
| Registered Community | Early-access hosted service | Free registration | Hosted adapted catalog, official community catalog when available, hosted collection updates, registered local enhancer script, signed device proof for hosted calls, and early-access catalog access. |
| Team Free | Implemented client MVP | Registered team | Manual approval by default, master/admin approval, members/pending/reject/revoke, dry-run sync, team collection listing, local audit events, and up to 10 instances when enforced server-side. |
| Pro / Team | Planned paid tier | Registered paid plan | Larger teams, dashboard, private packs, richer collection assignment, longer auto-approval windows, and support. |
| Enterprise | Planned enterprise tier | Contracted enterprise plan | Private registry, Enterprise Skill Lock, signed policy, managed update channels, SSO/on-prem/VPC options later, and enterprise support. |

## Boundaries

Community Core stays usable offline. No registration means no official hosted skill updates and no official community catalog access, but local skills remain usable.

Hosted archive downloads are SHA256-verified before extraction today. Cryptographic signature verification is planned and must not be claimed as implemented until the client enforces it.
