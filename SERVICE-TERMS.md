# Hosted Service Access Policy

This file describes access to AI4sale-hosted Unlimited Skills services. It does not change the MIT license for the repository source code.

## MIT source code

The local Unlimited Skills core in this repository is licensed under the MIT License. The MIT license remains in [LICENSE](LICENSE).

## Hosted services

The following are hosted services, not part of the unrestricted MIT source-code grant:

- official hosted adapted-skill catalog;
- hosted `community-skills` catalog and submissions;
- adapted collection update stream;
- registered local skill enhancement scripts;
- signed hosted collection archives;
- hosted dashboards, support, cloud sync, team skill sync, Enterprise Skill Lock, private registries, and marketplace features.

Access to hosted services requires a registered installation and a valid hosted-service token.

## Registered installations

A registered installation is identified by an install id and a hosted-service token stored locally in:

```text
~/.unlimited-skills/registration.json
```

The token is used only to access hosted services. It is not required for local/offline use of the MIT core.

## No investment rights

Hosted-service access, registration keys, donations, subscriptions, or paid plans are not securities, investments, equity, loans, or ownership interests. They do not provide any right to a share of AI4sale, revenue, profit, assets, voting rights, governance rights, or future fundraising.

## No refund promise

Unless a separate written agreement says otherwise, donations and voluntary support payments are non-refundable.

Hosted-service plans, endpoints, availability, and commercial terms may change independently from this repository's MIT source license.
